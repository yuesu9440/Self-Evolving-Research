# 亲水胶体及高分子复合系统分子动力学模拟参考数据集

## 目录结构
* `stress_strain_data.csv`: PVA-MXene 与 CS-CMC 水凝胶在 300K 下的拉伸应力-应变响应基准曲线。
* `rdf_data.csv`: 体系核心非共价作用（氢键与静电络合）的径向分布函数。
* `msd_data.csv`: 水分子在不同网络结构中的均方位移，用于校准扩散系数。
* `density_vs_temperature.csv`: 模拟体系的密度-温度依赖性，用于验证力场密度的准确性。
* `lammps_hydrogel.in`: 适用于全原子高分子系统的 LAMMPS 模拟控制输入模板。

## 使用建议
1. 在自己运行模拟后，可使用 Origin 绘图软件将实验/新模拟的数据与本目录下的 `.csv` 文件进行重叠对比。
2. 若新模拟的 $g(r)$ 峰位置相差超过 0.2 Å，或密度误差超过 5%，请优先检查初始模型的“含水量”是否与参考系统一致，以及电荷赋予（Partial Charges）是否正确。
